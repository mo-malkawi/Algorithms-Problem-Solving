# Algorithms & Problem Solving

C++ solutions to 200+ algorithmic problems across 4 levels, following the ProgrammingAdvices roadmap by Mohammed Abu-Hadhoud.

## Dependencies
Some levels use personal utility libraries available at [cpp-personal-libraries](link here):
- **Level 3**: uses `MyLib.h`
- **Level 4** (Problem #20 onwards): uses `DateTimeLib.h`

## Level 1: Fundamentals
50 problems covering basic input/output, conditionals, loops, and math operations.

Topics: arithmetic, geometry (areas of shapes), grade systems, ATM PIN simulation, loan calculations, date/time conversions.

## Level 2: Intermediate
50 problems covering number theory, patterns, strings, arrays, and math utilities.

Topics: prime numbers, palindromes, encryption, random number generation, array manipulation, and implementing math functions from scratch (`MyABS`, `MyRound`, `MyFloor`, `MyCeil`, `MySqrt`).

## Level 3: Matrices & Strings & Files
51 problems covering 2D arrays, string manipulation, and file handling.

Topics: matrix operations (transpose, multiply, identity, sparse), string processing (trim, split, join, replace), recursion (Fibonacci), and a mini bank client system using file I/O.

Uses `MyLib.h`.

## Level 4: Date & Time
65 problems focused entirely on date and time logic built from scratch.

Topics: leap years, calendars, date arithmetic (add/subtract days), date comparison, period overlap, vacation calculations, and date validation.

Problems #20 onwards use `DateTimeLib.h`.

## Technologies
C++, Arrays, Matrices, Strings, File Handling, Recursion
